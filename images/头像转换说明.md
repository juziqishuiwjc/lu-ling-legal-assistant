# 头像图片使用说明

## 已生成的SVG文件

### 1. user-avatar.svg（用户头像）
- 设计：简洁的人物图标
- 颜色：紫色渐变（#667eea → #764ba2）
- 风格：现代简约

### 2. assistant-avatar.svg（助手头像）
- 设计：法律天平图标
- 颜色：蓝色渐变（#4A90E2 → #357ABD）
- 风格：专业法律风格

## 转换为PNG的方法

### 方法1：在线转换（推荐）
1. 访问以下任意网站：
   - https://convertio.co/zh/svg-png/
   - https://cloudconvert.com/svg-to-png
   - https://www.aconvert.com/cn/image/svg-to-png/

2. 上传SVG文件
3. 选择输出尺寸：80 x 80 像素
4. 下载转换后的PNG文件

### 方法2：使用设计工具
1. 打开 Figma、Sketch 或 Adobe Illustrator
2. 导入SVG文件
3. 导出为PNG，尺寸设为 80x80px

### 方法3：直接使用SVG（微信小程序支持）
微信小程序原生支持SVG，可以直接修改 `index.wxml` 中的图片标签：

```xml
<!-- 将 -->
<image class="avatar user-avatar" src="/images/user-avatar.png" mode="aspectFit"></image>

<!-- 改为 -->
<image class="avatar user-avatar" src="/images/user-avatar.svg" mode="aspectFit"></image>
```

## 上传到图床的步骤

1. 将SVG转换为PNG（80x80像素）
2. 选择图床服务：
   - 国内：https://sm.ms/、https://imgurl.org/
   - GitHub（作为仓库图片）
   - 自有服务器

3. 上传图片获取URL
4. 修改 `pages/index/index.wxml` 文件：

```xml
<!-- 用户头像 -->
<image class="avatar user-avatar" src="https://your-image-host.com/user-avatar.png" mode="aspectFit"></image>

<!-- 助手头像 -->
<image class="avatar assistant-avatar" src="https://your-image-host.com/assistant-avatar.png" mode="aspectFit"></image>
```

## 预览效果

两个头像都使用了：
- 200x200 的SVG画布
- 渐变色背景
- 白色图标
- 圆形裁剪显示（通过CSS border-radius实现）

在小程序中会显示为80x80的圆形头像，效果简洁专业。
