## 🔧 如何添加新功能

### 步骤1: 创建页面文件
```bash
pages/new-function/
├── new-function.wxml
├── new-function.wxss
├── new-function.js
└── new-function.json
```

### 步骤2: 注册路由 (app.json)
```json
"pages": [
  "pages/new-function/new-function",
  ...
]
```

### 步骤3: 配置模块 (pages/home/home.js)
```javascript
{
  id: 'new-function',
  name: '新功能名称',
  icon: '🎯',
  color: 'linear-gradient(...)',
  page: '/pages/new-function/new-function'
}
```
