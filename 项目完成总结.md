## 💻 使用指南

### 1. 启动项目
```bash
# 在微信开发者工具中打开
xiaochengxu/

# 1. 导入项目
# 2. 配置AppID(或使用测试号)
# 3. 点击"编译"
```

### 2. 配置API Key
在 `app.js:258` 配置GLM-4 API Key：
```javascript
glmConfig: {
  apiKey: 'YOUR_API_KEY_HERE', // 智谱AI API Key
  baseUrl: 'https://open.bigmodel.cn/api/paas/v4/chat/completions'
}
```

### 3. 测试路径
```
首页 → AI问答 → 输入问题 → 查看回复
首页 → 诉讼费用 → 输入100000 → 查看计算结果
首页 → 工伤赔偿 → 选择十级伤残 → 查看赔偿明细
首页 → 级别管辖 → 查看各级法院管辖标准
```
