## 💡 使用说明

### 1. 启动小程序
打开小程序后,直接进入功能导航首页,可以看到20个功能模块。

### 2. 使用AI问答
点击"AI问答"模块,进入原有的聊天界面,可以:
- 输入法律问题
- 查看剩余次数
- 会员可复制回答内容

### 3. 使用诉讼费用计算器
点击"诉讼费用"模块,进入计算器:
1. 选择案件类型(财产案件、离婚案件等)
2. 输入诉讼标的额
3. 点击"计算费用"
4. 查看计算结果和费用标准

### 4. 其他功能
点击其他功能模块,会提示"功能开发中"。

## 🛠️ 技术实现

### 功能模块数据结构
```javascript
{
  id: 'ai-chat',           // 唯一标识
  name: 'AI问答',          // 显示名称
  icon: '🤖',              // emoji图标
  color: 'linear-gradient(...)',  // 渐变色背景
  page: '/pages/index/index'      // 跳转路径
}
```

### 路由跳转逻辑
```javascript
handleFunctionTap(e) {
  const id = e.currentTarget.dataset.id
  const target = functions.find(item => item.id === id)

  if (target && target.page) {
    // 跳转到对应页面
    wx.navigateTo({ url: target.page })
  } else {
    // 提示"功能开发中"
    wx.showToast({ title: '功能开发中', icon: 'none' })
  }
}
```

## 🚀 如何添加新功能

### 步骤1: 创建页面文件
```bash
pages/new-function/
├── new-function.wxml
├── new-function.wxss
├── new-function.js
└── new-function.json
```

### 步骤2: 在 app.json 中注册路由
```json
"pages": [
  "pages/home/home",
  "pages/new-function/new-function",  // 添加新路由
  ...
]
```

### 步骤3: 在 home.js 中添加模块信息
```javascript
{
  id: 'new-function',
  name: '新功能名称',
  icon: '🎯',
  color: 'linear-gradient(...)',
  page: '/pages/new-function/new-function'  // 配置页面路径
}
```
