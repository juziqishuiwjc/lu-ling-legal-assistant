# 庐陵吉成法答 - 法律工具微信小程序

这是一个面向法律从业者和普通用户的法律工具微信小程序，提供19个实用的法律计算器和1个AI智能助手。

## 小程序简介

- **名称**：庐陵吉成法答
- **开发者**：江西吉泰律师事务所 王吉成律师
- **定位**：法律工具小程序 + AI法律咨询助手

## 功能特点

### 核心功能模块（20个）

| 序号 | 功能 | 说明 |
|------|------|------|
| 1 | 王律师助手 | AI智能法律咨询（基于GLM-4.6） |
| 2 | 诉讼费用 | 根据《诉讼费用交纳办法》计算案件受理费 |
| 3 | 利息/违约金 | 计算一般利息和违约金 |
| 4 | 迟延履行利息 | 计算民事诉讼法规定的迟延履行期间债务利息 |
| 5 | 民间借贷利息 | 民间借贷利息计算（支持LPR基准） |
| 6 | 律师费 | 各类案件律师服务费计算参考 |
| 7 | 破产管理人报酬 | 破产案件管理人报酬计算 |
| 8 | LPR查询 | 全国银行间同业拆借中心贷款市场报价利率查询 |
| 9 | 日期推算 | 法律期限计算（立案时效、上诉期等） |
| 10 | 交通事故 | 交通事故赔偿项目计算 |
| 11 | 工伤赔偿 | 工伤保险待遇计算 |
| 12 | 经济补偿金 | 劳动合同解除/终止经济补偿金计算 |
| 13 | 带薪年休假 | 职工带薪年休假天数及补偿计算 |
| 14 | 退休年龄计算 | 渐进式延迟法定退休年龄推算（2025年新规） |
| 15 | 小额诉讼限额 | 小额诉讼案件标的额标准查询 |
| 16 | 级别管辖 | 各级法院管辖标的额标准 |
| 17 | 仲裁费用 | 仲裁委员会仲裁费用计算 |
| 18 | 基准利率 | 中国人民银行基准利率（开发中） |
| 19 | 民事案由 | 民事案件案由查询 |
| 20 | 常用年度数据 | 各省年度法律数据（赔偿标准等） |

## 开发工作流

### 修改功能模块

每个计算器模块对应 `pages/` 下的独立文件夹，包含：
- `.js` - 页面逻辑
- `.wxml` - 页面结构
- `.wxss` - 页面样式
- `.json` - 页面配置

### 添加新功能

1. 在 `pages/` 下创建新文件夹
2. 在 `app.json` 的 `pages` 数组中注册新页面
3. 在 `pages/home/home.js` 的 `functions` 数组中添加入口配置

## 上线流程

1. 代码上传（微信开发者工具）
2. 提交审核（填写审核信息）
3. 等待审核（1-7个工作日）
4. 审核通过后发布
