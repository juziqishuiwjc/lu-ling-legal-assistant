## 📱 使用指南

### 基本对话
1. 在输入框输入法律问题
2. 点击"发送"按钮
3. 等待王律师的回复
4. 查看回复末尾的联系方式

### 连续对话
- 支持上下文对话
- 可追问相关问题

## 🔧 技术细节

### API 调用优化
- 简化系统提示词，提升响应速度
- 仅保留最近1条对话历史（从10条优化至1条）
- 单条消息长度限制500字符（从2000字符优化）
- max_tokens设置为2000（从3000优化）
- 请求大小减少约80%，显著提升响应速度

### UI 优化
- 使用 `animation` 实现淡入效果
- 使用 `transition` 实现平滑过渡
- 使用 `backdrop-filter` 实现毛玻璃效果
- 使用 `linear-gradient` 实现渐变背景
